More Mathjax goodies

Use specific macros inside your latex code to style specific elements in your equation and apply fragments!
  • \texclass{any_class}{element} to style an element with any_class
  • \fragment{index}{element} to make the element an appearing fragment with data-fragment-index=index
  • \fragapply{class optionalIndex}{element} to change the style of the element with optional data-fragment-index = optionalIndex
$s= \frac{\texclass{info}{\alpha_0}}{4} + \sqrt{\fragapply{danger 0}{\frac{\sum(x_{i}-\bar{x})^2}{n-1}}} - \int_{a}^{b} \fragapply{smaller 1}{x^2} \fragment{0}{dx}$

Animations are supported too!

Classes can be added and removed using data-animate.
Eg.: data-animate="A + B - C" adds .A, .B and removes .C.
All animations from Animate.css are bundled!
Automatically triggered on fragment descendants!
Speed can be controlled with .infinite or data-duration.
Animations can be delayed with data-delay!
